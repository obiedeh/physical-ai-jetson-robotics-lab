# Synria 6DOF Arm

Isaac-side robot assets for the Synria 6DOF arm.

- `synria_6dof_arm.urdf`: generated working URDF for Isaac import.
- `synria_6dof_arm.usd` / `synria_6dof_arm.usda`: imported Isaac USD robot outputs.
- `configuration/`, `payloads/`, `Textures/`: Isaac-generated support assets.
- `source_urdf/`: xacro source copy used to regenerate the URDF.

Canonical editable ROS 2 source lives in:

`ros2_ws/src/synria_arm_description/urdf/`
