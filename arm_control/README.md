# Robotic Arm Control

This track targets the Synria AI Robotic Arm Kit, a 6DOF arm with 650 mm reach,
1 kg payload, ROS 1 / ROS 2 and MoveIt support, LeRobot / ALOHA-compatible
learning workflows, and the Synria C10 wrist camera.

Planned work:

- kinematic model documentation
- OpenUSD robot asset references
- Isaac Lab reaching and pick-place tasks
- ROS 2 / MoveIt 2 bring-up
- C10 camera calibration and eye-in-hand perception
- LeRobot / ALOHA-style demonstration data capture
- real-arm calibration procedure
- safety-gated deployment scripts
